Step 2 · Run the workflow

Same flow as the sample. Real CSV.

  1. Open Google Classroom → your class → Grades → click the gear → Download all grades as CSV.
  2. Open the file in Google Sheets for a quick eyeball — does anything look obviously off? (No need to fix; the Gem will.)
  3. Drag the CSV into your Classroom Data Analyzer Gem. Read its Phase 1 cleanup pass. Reply go.
  4. Answer the three Phase 2 questions. Lock in your audience above.
  5. The Gem builds. Save the HTML as my-dashboard.html, with ui-library.css next to it.
  6. Open the dashboard locally. Iterate with the Gem if anything is off — "swap the chart to a line", "try the parents voice", "flag students near the proficient line."
